CSS

계단식 시트(CSS)는 HTML과같은 태그 언어로 작성된 문서의 프레젠테이션을 설명하는 데 사용되는 스타일 시트 언어입니다. [1] CSS는 HTML 및 자바 스크립트와함께 월드 와이드 웹의초석 기술입니다. [2] CSS는 레이아웃, 색상및 글꼴을포함한 프레젠테이션 및 콘텐츠를 분리할 수 있도록 설계되었습니다. [3] 이러한 분리는 콘텐츠 접근성을향상시키고, 프레젠테이션 특성의 사양에 대한 유연성과 제어를 제공하고, 구조 콘텐츠의 복잡성과 반복을 줄이는 별도의 .css 파일에 관련 CSS를 지정하고 파일을 공유하는 페이지 로드 속도를 높이기 위해 .css 파일을 캐시할 수 있도록 하여 서식을 공유할 수 있습니다. 서식과 콘텐츠를 분리하면 화면, 인쇄, 음성 기반 브라우저 또는 화면 판독기, 점자 기반 촉각 장치 등 다양한 렌더링 메서드에 대해 동일한 태그 페이지를 다른 스타일로 표시할 수 있습니다. CSS에는 모바일 장치에서 콘텐츠에 액세스하는 경우 대체 서식에 대한 규칙도 있습니다. [4] 이름 계단식은 지정된 우선 순위 구성표에서 유래하여 두 개 이상의 규칙이 특정 요소와 일치하는 경우 적용되는 스타일 규칙을 결정합니다. 이 계단식 우선 순위 체계는 예측 가능합니다. CSS 사양은 W3C(월드 와이드 웹 컨소시엄)에 의해 유지됩니다. 인터넷 미디어유형(MIME 타입)은 RFC 2318(1998년 3월)에 의해 CSS에 사용하기 위해 등록되어 있습니다. W3C는 CSS 문서에 대한 무료 CSS 유효성 검사 서비스를 운영합니다. [5] text/css HTML 외에도 다른 태그 언어는 XHTML, 일반 XML, SVG및 XUL을포함한 CSS의 사용을 지원합니다.
