JavaScript

자바 스크립트 (/dý dý ɑːvâésskrpât/),[6] 종종 JS로축약, ECMA스크립트 사양에 부합하는 프로그래밍 언어입니다. [7] 자바 스크립트는 높은 수준,종종 그냥 시간 컴파일,멀티 패러다임입니다. 곱슬 브래킷 구문, 동적 타이핑, 프로토타입 기반 개체 지향및 일류 기능이 있습니다. HTML 및 CSS와함께 JavaScript는 월드 와이드 웹의핵심 기술 중 하나입니다. [8] 자바 스크립트는 대화 형 웹 페이지를 활성화하고 웹 응용 프로그램의 필수적인 부분입니다. 대부분의 웹 사이트는 클라이언트 측 페이지 동작에 사용하며[9] 모든 주요 웹 브라우저에는 전용 JavaScript 엔진이 실행됩니다. 다중 패러다임 언어인 JavaScript는 이벤트 중심의 기능및 필수 프로그래밍 스타일을지원합니다. 텍스트, 날짜, 일반 식,표준 데이터 구조및 문서 개체 모델(DOM)으로 작업하기 위한 응용 프로그램 프로그래밍 인터페이스(API)가 있습니다. ECMAScript 표준에는 네트워킹, 저장소또는 그래픽 시설과 같은 입력/출력(I/O)이 포함되어 있지 않습니다. 실제로 웹 브라우저 또는 기타 런타임 시스템은 I/O용 JavaScript API를 제공합니다. 자바 스크립트 엔진은 원래 웹 브라우저에서만 사용되었지만 지금은 노드.js 및 Deno와같은 다른 런타임 시스템의 핵심 구성 요소입니다. 이러한 시스템은 서버를 빌드하는 데 사용되며 다양한 응용 프로그램을 만들기 위해 전자 및 코르도바와같은 프레임워크에도통합되어 있습니다. 언어 이름, 구문및 각 표준 라이브러리를포함하여 JavaScript와 Java사이에 유사점이 있지만 두 언어는 뚜렷하고 디자인이 크게 다릅니다.
